## ▶️ Primeros Pasos

Sigue este flujo para usar la plataforma correctamente:

## 🔐 1. Iniciar sesión

- Accede con tu cuenta institucional o **Google**
- El sistema validará tu acceso automáticamente  

## ⏳ 2. Reservar o entrar en cola

- Puedes **reservar una sesión** o unirte a la **cola**
- Espera tu turno para acceder al hardware  

## 🧪 3. Ingresar al laboratorio

- Cuando sea tu turno, entra a la plataforma  
- Se habilitarán los módulos disponibles  

## 🚫 Acceso restringido

- Si **no has iniciado sesión**, algunas páginas estarán bloqueadas  
- Si **ya iniciaste sesión**, todo el contenido se habilita automáticamente  

## 🖥️ 4. Control remoto (noVNC)

- Accede a la pantalla remota del dispositivo  
- Interfaz basada en navegador (noVNC)  

**Credenciales:**
- Password: `123456`

## ✅ Recomendaciones

- No abandones tu turno activo  
- Cierra sesión al finalizar  
- Respeta el tiempo asignado
