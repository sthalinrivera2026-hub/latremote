## 🚀 Introducción
Plataforma web para interactuar con **hardware real** de forma remota.
Permite trabajar directamente desde el navegador con:
- ESP32  
- Arduino  
- Jetson Nano  
- Raspberry Pi  
## 🎯 Objetivo
- Realizar prácticas reales sin estar físicamente en el laboratorio  
- Controlar sensores y actuadores en tiempo real  
- Visualizar resultados mediante cámara en vivo  
- Gestionar el acceso de usuarios mediante colas  
## 🔐 Acceso
- Inicio de sesión con cuenta institucional o Google  
- Sistema de **cola de usuarios** para evitar conflictos  
- Tiempo de uso limitado por sesión  
## 🧩 Módulos
- **ESP32** → Conectividad WiFi e IoT  
- **Arduino** → Sensores y electrónica básica  
- **Jetson Nano** → IA y visión por computadora  
- **Raspberry Pi** → Sistema Linux y servicios
